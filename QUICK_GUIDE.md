## ⚡ 3-STEP DEPLOYMENT

### STEP 1: Push to GitHub (30 seconds)
```bash
git add .
git commit -m "ESPON v1.0 Complete"
git push origin main
```

### STEP 2: Deploy on Vercel (2 minutes)
Go to [vercel.com](https://vercel.com) and:
1. Import your repository
2. Set **Root Directory: web**
3. Add environment variables (see below)
4. Click **Deploy**

### STEP 3: Test & Launch (2 minutes)
Visit your app at: `https://your-app.vercel.app/login`

**Total time: 5 minutes** ⏱️

## 🎓 QUICK TUTORIAL

### For Agents:
1. Login at `/login`
2. Click **"Punch In"** to start work
3. Go to **Customers** → **Add Customer**
4. Go to **Calls** → **Log Call**
5. Go to **Orders** → **Create Order**
6. Check **Dashboard** for stats
7. Click **"Punch Out"** to end work

### For Admins:
1. Login at `/login`
2. View **Dashboard** for live status
3. Go to **Agents** → Set monthly targets
4. Go to **Customers** → Manage master DB
5. Go to **Leave** → Approve requests
6. Go to **Reports** → Export data
7. Go to **Settings** → Configure system
